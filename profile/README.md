## What is AI4Change?

In 2005, [Playing for Change](https://playingforchange.com) proved something radical: **when you remove the barriers, ordinary people create extraordinary things.** They connected musicians across continents вҖ” a guitarist in New Orleans, a drummer in Mumbai, a choir in Soweto вҖ” and mixed their recordings into songs no single studio could produce.

**AI4Change does the same thing. But with solutions, not songs. And with AI, not instruments.**

A farmer in Kenya uses AI to predict crop disease. A teenager in Sao Paulo builds an air quality monitor. A retired teacher in rural India creates a literacy tool. Each works independently, each uses AI as their multiplier вҖ” and their contributions compose into **open-source solutions the world can use.**

We call each project a **"song."** Every contributor is a musician. The code is the melody. And the world is the audience.

## What Makes This Different

</div>

| What Exists | What We Do Instead |
|:------------|:-------------------|
| **Hackathons** build for 48 hours, then forget | We **build to last** вҖ” continuous, open, evolving |
| **Nonprofits** need funding cycles and bureaucracy | We need a **GitHub account and willingness** |
| **AI ethics orgs** write papers about what AI *should* do | We **build the things** AI should be used for |
| **Open source projects** start with a solution looking for a problem | We **start with the problem** and let the community shape the solution |
| **Corporate CSR** is top-down and image-driven | We're **bottom-up** and community-driven |
